# pdoc-parser

Parses e:cue pdoc xml files to a javascript object.

## Installation

```js
npm install --save pdoc-parser
```

## Usage


```js
const Pdoc = require('pdoc-parser');
const fs = require('fs');

const pdocXml = fs.readFileSync('test-data/test.pdoc').toString();
const patch = Pdoc(pdocXml);
```
